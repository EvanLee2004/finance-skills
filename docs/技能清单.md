## 约定

- **命名**：技能子目录用简短英文 / 拼音 + 说明，如 `skills/find-receipt/`、`skills/split-by-sales/`。
- **接口**：每个技能后续应自带一份 `README.md`，说明输入参数、输出格式、调用示例。
- **只读铁律**：任何技能都不得写回源数据目录；产物只落 `outputs/`，并带时间戳防覆盖。
- **新增技能流程**：建子目录 → 写技能 README → 在本表登记一行 → 让 agent 可发现调用。
