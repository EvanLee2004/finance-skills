# 应收 all 按销售人员拆分

把财务部手工"把应收 all 按销售拆成一人一份"自动化。它是 **receivables-merge（合并）的下一棒**：合并出 all → 本技能按销售拆 → （后续）发给各销售填回。

## 你（agent）该怎么干

**核心原则：用它的是财务同事（如赵亮晶）。她给一张应收 all，你拆好、对账、交付——别让她看命令、看 sheet 名。** 拆分比合并简单：**归属在合并那步已经做完了**（all 里的销售人员已是最终名），本技能只按这个名字分组，所以**很少需要问她**。

### ⭐ 人在环：拆分该问的很少
- **自己干**：找输入 all、按销售人员分组、套模板（下拉/配色/冻结）、生成每人一份、对账。
- **只在这几种情况问她**：
  1. **对账没对上**（分出+空名+忽略 ≠ 输入）→ 别交付，报给她："对账差了 N 行，先别发，我查下。"
  2. **有"销售人员为空"的行**（进了 `_销售人员为空_请人工处理.xlsx`）→ 问她："有 N 行没销售人员，你看下归谁？"
  3. **冒出没见过的"X-高美杰"复合名 / 新的坏账桶名** → 问她这个 X 的高美杰单要不要单独成 GM 订单 sheet（按需补进 `config/拆分规则.md`）。
- **别问的**：拆分本身、模板样式、各人多少行——跑完一句话汇总即可。

### 1. 找输入（应收 all）
**用户说文件在哪（"在 ./数据/""在桌面那个文件夹"），就直接用那个路径**——绝对路径传给 `--input`，别傻等文件出现在 `工作区/input/`。也可能是 receivables-merge 刚产出的那张 all。只有用户没说、也没给具体文件时，才去 `工作区/input/` 取最新 xlsx。

### 2. 跑拆分
```bash
# 把 <本skill目录> 换成这份 SKILL.md 所在文件夹的绝对路径
python3 "<本skill目录>/scripts/split.py" --input <应收all绝对路径> --date <月日如0604>
```
- **`--out-dir` 可省**：给了具体 all → 输出放它**同目录**的 `拆分_<日期>/`（她找得到）；没给、自动从 `工作区/input/` 取的 → 落 `工作区/output/拆分_<日期>/`（输入输出不混）。**若源文件在一个"输入"文件夹里、不想让拆分结果和源混在一起，就显式给 `--out-dir <一个干净目录>`** 当交付夹。
- `--date` 给当期月日（如 `0604`，进文件名和表头日期前缀）；**不给会自动从 all 文件名取**（`2026.6.4应收all` → `0604`），文件名没日期才退回当天（并告警）——所以一般不用手动传。
- 脚本自动：认数据 sheet → 对齐 17 列 → 按销售人员分组（坏账桶忽略 / X-高美杰归 X / 于占国的进 GM订单 sheet）→ 每人一份带下拉模板、**按客户名称排好（同一客户的单归拢在一起，亮晶姐 0620 定的排序）** → **对账**。

### 3. 跑完收尾（人话，给摘要）
看脚本输出的对账行：
- **对账 ✓ 对得上、没空名** → 一句话带过："拆好了，15 位销售、各一份，在 <目录>，对账没问题。"
- **对账 ✗ 或有空名** → 才提她（见人在环）。
- 别把 15 个文件清单/每人行数全甩给她，她要的是"拆好了、对上了、在哪"。

### 3.5 ⭐ 像同事一样收尾（数字员工三件事 · 主动但克制）
拆分简单、平时一句话带过即可，但有这几种"不对劲"要像同事一样主动点出来：
1. **环比/异常小结**：若知道上次拆分情况，比一句"这次 **N 位销售**（上次 M，**多/少了谁**）"；某人单量突然暴增/归零也提一句。
2. **拿不准就提具体问题**：冒出**没见过的"X-高美杰"复合名**或**新坏账桶名**——别默默并表或忽略，问她"这个 X 的高美杰单要不要单独成 GM 订单 sheet？这个名要不要进忽略名单？"
3. **本岗位该主动操心啥**：**对账没对上**（绝不能发，先查）、**有空销售名行**（进了待人工文件，问归谁）、**某销售一份单都没有**（是不是漏了/名字对不齐）。

## 会变的东西在哪（改表不改码）
- **拆分规则**（坏账桶忽略名单、GM 单独成 sheet 的接手人）→ `config/拆分规则.md`（人维护）。
- 模板（17 列表头、下拉选项、配色、列宽）在脚本里（亮晶姐模板，稳定；要改在 `scripts/split.py` 顶部）。

## 验收口径
用真实 all（如 `2026.6.4应收all`，3243 行）跑：应得 **15 位销售**、于占国含 **GM订单 74 行**、高美杰1 **忽略 10 行**、**对账 3233+0+10=3243 ✓**。换别的 all：看对账对不对得上、有没有空名，即可信。
