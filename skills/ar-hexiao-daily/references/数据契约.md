# 数据契约

## 输入

| 文件 | 谁产生 | 关键字段 |
|---|---|---|
| `01_智云导出/回款记录_*.xlsx` | 她导出 | AR / 核销日期 / 到账日期 / 到账金额原币 / 手续费 / 原币币种 / **回款类型** / 核销状态 / 开票客户 |
| `01_智云导出/回款核销对账_*.xlsx` | 她导出 | AR / SO / SOD / 回款额原币·本币 / 交付额 / 核销日期 / 客户名称 / 结算周期 |
| `01_智云导出/核销明细_*.xlsx` | 她导出（预存） | 回款记录NUM / **核销日期** / 本次核销金额 / 币种 / 汇率 |
| `02_我的表副本/盈亏核算表*.xlsx` | 她另存 | 「明细」：新智云单号(SO) / 实收金额(**存 SOD**) / 应收 / 计提 / 回款明细 / 是否结账 / 收款时间 / 收款方式 |
| `02_我的表副本/到账流转表*.xlsx` | 她另存 | 日期 / 公司名称·汇款人 / 金额 / 收款形式 / 单号 / 是否更新应收款 |
| 离线夹具 JSON | 测试 | `std` / `day` / `fields_duizhang` / `fields_huikuan` |

列名一律走 `config/列名别名.json` 模糊匹配；找不到 → 非 0 退出并列出实际表头。

## 输出

| 文件 | 谁产生 | 内容 |
|---|---|---|
| `03_台账/挂账台账.xlsx` | 脚本 | AR / SO列表 / 挂起日 / E码 / 原因 / 重扫次数 / 最近重扫日 / 状态 |
| `04_产出/判定结果_YYYYMMDD.json` | classify | auto / hold / exception + counts |
| `04_产出/今日工作清单_YYYYMMDD.xlsx` | build_worklist | 三页签：今天能填 / 挂账待办 / 异常 |
| `04_产出/运行报告_*.txt` | 脚本 | 计数、E 码分布（无客户名金额明细） |
| `04_产出/收入汇总_*.xlsx` | extract_income | 收入清单 |

### 今天能填（每行）

AR / SO / SOD / 客户(打码) / **怎么找到这行（按 SO 筛选）** / 五列应填 / **当前值** / 判定依据。  
**禁止用行号定位。**
