### 《核销日清》→ `今日清单` 每行

**第一列＝单号（SOD，无 SOD 时用 SO）**，**第二列＝状态**（今天要填 / 已填过·跳过 /
冲突·需你定 / 挂账待办 / 异常），第三列＝怎么办；后面是 SO / AR / 客户(打码) / 本次金额 /
五列应填 / 五列当前 / 差异(一眼扫) / **怎么找到这行（按 SO 筛选）** / 流转表定位 / 判定依据 / 码。

**禁止用行号定位**（她随时插行，行号隔天失效）。
