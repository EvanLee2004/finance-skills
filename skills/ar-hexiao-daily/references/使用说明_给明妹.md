# 应收核销日清 · 使用说明（给明妹）

> 读者是出纳，不是程序员。照着做就行。

---

## 1. 一次性准备

1. 电脑上已有本技能文件夹（同事会帮你放好）。  
2. 需要 Python 和 `openpyxl`（同事装好；你一般不用管）。  
3. 记住工作区长这样：

```
工作区/
  01_智云导出/     ← 你从智云导出的表放这里
  02_我的表副本/   ← 盈亏表、流转表「另存为」副本放这里（原件别直接丢进来）
  03_台账/         ← 电脑自动维护的挂账台账
  04_产出/         ← 每天的清单和报告从这里拿
```

4. 第一次可先让同事帮你跑一遍「盘点」，看文件夹是否认得出。

---

## 2. 每天怎么做（5 步）

1. **导出**  
   在智云按「核销日期 = 昨天」筛，导出回款记录（需要时再导出对账表、预存核销明细）。放到 `01_智云导出/`。  
   （截图位：智云筛选界面 / 导出按钮）

2. **另存副本**  
   把今天要对照的「盈亏核算表」「到账流转表」**另存为副本**，放进 `02_我的表副本/`。  
   **不要**让程序改你正在用的原表（第一版程序也改不了你的原表）。

3. **说一句话**  
   对同事或助手说：「跑昨天的核销」或「做核销清单」。  
   他会按顺序跑：盘点 →（如需）收入提取 → 核销判定 → 生成清单 → 挂账重扫。

4. **拿清单**  
   打开 `04_产出/今日工作清单_日期.xlsx`。  
   三个页签：`今天能填` / `挂账待办` / `异常`。

5. **处理挂账和异常**  
   按清单说明自己在表里改；挂着的过几天交付齐了再说「重扫挂账」。

---

## 3. 清单怎么看

### 页签「今天能填」

电脑认为可以回填的笔。每一列：

| 列 | 意思 |
|---|---|
| AR / SO / SOD | 回款号、订单号、结算号 |
| 客户(打码) | 只显示打码，保护隐私 |
| 怎么找到这行 | **按 SO 号在盈亏表筛选**，不要按行号（表会插行，行号会变） |
| 应填_计提 / 回款明细 / 是否结账 / 收款时间 / 收款方式 | 建议填什么 |
| 当前_* | 你表里现在是什么；对不上说明表变了，别盲填 |
| 判定依据 | 一句话为什么这样判 |

### 页签「挂账待办」

暂时不能填盈亏的（比如分笔、还没交付、跨年老单）。先可在流转表写单号，盈亏等条件具备再补。

### 页签「异常」

数据有问题（对不上账、超额、缺汇率等），需要你或销售先查清楚。

---

## 4. 每个异常原因怎么办

（大白话，不记代号）

| 你看到的话 | 你该做什么 | 找谁 |
|---|---|---|
| 分笔回款，系统没有拆到每张单的金额 | 先挂着；流转表可只写单号；等回满或销售补清再补盈亏 | 销售 |
| 盈亏表里还没有这张单 | 等多交付贴进明细，再说重扫挂账 | 自己 / 月初流程 |
| 往年的单，今年表里没有 | 按老单人工流程，或查往年表 | 自己 / 主管 |
| 核销金额比到账还多 | 先别填，查系统是不是录错 | 销售 |
| 金额差了一点点也对不上 | 先挂着，人工看是不是尾差（规则还没定宽松阈值） | 自己 |
| 外币缺汇率 | 补当天中行牌价后再跑 | 自己 / 同事 |
| 没有单号或已作废 | 跳过；无单号问销售 | 销售 |
| 这张单在表里有好几行 | 你指定记哪一行再填 | 自己 |
| 带手续费，口径还没定 | 先挂着，等口径确认 | 明妹拍板 / 主管 |
| 到账流转对不上 | 查日记账有没有漏登 | 自己 |

---

## 5. 哪些它不会做

- **不替你录智云**（永不自动往系统写数据）。  
- **不改你的盈亏表、流转表原件**（第一版只出清单，你自己填）。  
- **分笔回款**判不了金额（大约一成），只能挂起。  
- **部分核销要加行**的，第一版不加行，只提示。  
- 不替销售做核销。

---

## 6. 出错了怎么办

1. 看弹窗或对话里的「ERROR」全文。  
2. 若提示「表头缺列…实际表头：…」——说明导出的列名和预设对不上，把报错截图给同事，别自己猜列。  
3. 日志/报告在 `04_产出/运行报告_*.txt`。  
4. 找谁：先找部署技能的同事（明昊侧）；业务口径找你自己或主管。

## 7. 常见问题

**Q：导出的表头和以前不一样？**  
A：程序按中文列名模糊认。认不出就报错停下，不会瞎猜。把实际表头发给同事加别名即可。

**Q：跑一半停了？**  
A：看报错。改完文件后从断的那步重跑；判定和清单可重复跑。

**Q：同一天跑两遍会不会重复？**  
A：挂账重扫同一天再跑不会叠次数；清单会覆盖生成。你自己往表里填时注意别重复累加——清单上有「当前值」可对照。

**Q：预存类准不准？**  
A：本地测试包几乎没有预存样例，程序已按规则写了（含按核销日期过滤），上线后要用你真实导出再验。同事不会假装「已经验过」。

**Q：收入提取那步？**  
A：把银行日记账给程序，它会挑出纯销售收入清单；空预填行会自动跳过；美元和人民币不会混加。
