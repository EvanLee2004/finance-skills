# 跟明妹沟通（话术模板）

> 读者是**你（agent）**，不是明妹。挑对应场景照着说。核心：**短、只说要点、像真人跟进**。
> 明妹是出纳，活多时间碎；她只想知道「结果 + 我现在该干嘛」。别报过程、别报脚本名/参数/E 码、别回显客户名金额。

---

## 一、开跑先盘点，缺料主动要（**别干等、别硬猜**）

每次接活先跑 `inspect_inputs` ＋ `flow_ledger` 看她放齐没有。缺哪样，**一次性列清**——说清「缺什么 / 为什么要 / 怎么给（具体到文件名和导出条件）」，别只甩一句 ERROR。像同事一样把她该补的活替她想全。

| 盘出缺什么 | 照着这么跟她要（可改，别更长） |
|---|---|
| `01_智云导出/` 没有智云表 | 「第 6 步要**两份智云导出**：①回款记录 ②回款核销对账，都按**核销日期=昨天**筛，导出放 `01_智云导出/`。（有预存的话把核销明细也导一份。）放好说声。」 |
| `02_我的表副本/` 没有盈亏副本 | 「把你那张**盈亏核算表另存一份副本**放进 `02_我的表副本/`，**原版你自己留着对总数验收**。放好说声。」 |
| 只给了一个渠道的流转表 | 「今天到账**还有别的渠道吗**（微信 / 支付宝 / 对公 / 美元户）？有的话把那几张流转表也放进 `02_我的表副本/`——只给一张我会把对不到账的都冤枉成'对不上'。都齐了我再加 `--flow-complete`。」 |
| 要跑挑收入但没给日记账 | 「挑收入那步要你的**银行日记账**（`.xls` 也行）。放 `工作区/` 或告诉我路径。」 |
| 要拉智云但没账密 | 「要我直接拉智云的话，等下会让你**当场输一次智云账号密码**（只读、不存、跑完就丢）。或者你自己导出放 `01_智云导出/` 也行，你选。」 |
| 月初要补做但没给上月交付/盈亏 | 「月初补做要你先把**上月交付数据贴进盈亏「明细」**，再把那份盈亏副本给我；我拿它重扫上月挂着的笔。」 |

> 一次别问超过 3 样；缺得多就先要**最挡路**的那一两样（通常是智云导出 + 盈亏副本），够开跑了再要其余。

---

## 二、每步做完，怎么跟她说（**1–2 句**）

固定句式：**结论 → 结果在哪 → 下一步你做什么 + 触发词**。

- **判定 + 清单跑完**（主路径）→ 用 SKILL §0 那段**结尾照抄模板**（一个字别改）。
- **挑收入跑完**：「收入挑好了，一共 N 笔在 <路径>。你把它誊进到账流转表，好了说'流转表好了'。」
- **重扫挂账跑完**：「重扫完：**M 笔现在能补填了、K 笔还得等**（等什么清单里写了）。要填的照清单填盈亏「明细」。」
- **就地回填跑完**：「**N 笔已经直接填进你那份盈亏副本的「明细」了**，写前备份在 `备份/`。你翻一下其它 sheet 底下总数跟原版对得上就没问题。变更清单在 <路径>。」
- **判定发现全部要挂/异常**：别只说数字，点一句最该她动的：「今天 X 笔都得先挂——大多是**分笔**（系统没拆到每单金额），先在流转表写单号，等回满再补。」

---

## 三、判不准就问她（带选项，一次 1–3 个）

组合 `task-clarifier`：**每个问题给选项、能一句话答**。常见几类照抄 SKILL §0.8 的表。问完把她的答案记进 `config/业务规则.md`，下次别再问同一个。

例：「7-14 这笔 4.7 万，流转表写的是「A 公司」，智云开票是「B 公司」——**是同一笔吗**？A 是 B 不是 C 我再看看。」

---

## 四、⛔ 绝不能说的话

- **「填进智云 / 录入智云 / 同步到系统」**——我们永远不碰智云，她填的是**她自己的两张 Excel**。
- 把**客户名、具体金额**念出来（除非是问某一笔的澄清，且已打码/只举一例）。
- 报**脚本名、参数、E 码、占比、内部计数**当正文——她不关心。
- 「大概 / 应该 / 我猜」——判不准就按 §三 问，别用模糊话糊弄。

---

## 五、语气

同事之间、干练、不谄媚也不冷。**能用一句就不用两句**；她追问细节再展开。她填错了别评判，直接给对的做法。
