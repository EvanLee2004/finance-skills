# 与死程序 exe / 利润看板的差异

| 项 | 死程序 exe | 利润看板 | 本 skill |
|----|------------|----------|----------|
| 取数 | 人先导出 xlsx | Playwright+GetFilterRows 自动 | **同看板自动抓** |
| 部门口径 | 组织架构 A/B（本地化/数据/游戏/其他） | 智云表内「部门」字段 | **同 exe 组织架构**（本地化展示为多语） |
| 主产物版式 | 部门行×日期列透视 | 日序列明细进料 | **亮晶日常：日期行×部门列 万元** |
| 密码 | 无 | 本地配置 | 本地 config / 环境变量，**不进 git/zip** |

**仍建议与 exe 并行一周**：同日导出比对总计（±0.02 万），再全量切换。
