# 财务技能包 · 来源与更新

> 本文件会随技能一起装到 opencode 的 `skills/` 目录旁（或包内）。  
> **更新财务技能时先看这里**——知道去哪个仓库 / 哪个包。

## 唯一源码仓（一个项目，不是每个 skill 一个仓）

| | |
|--|--|
| GitHub | https://github.com/EvanLee2004/finance-skills |
| Gitee（国内） | https://gitee.com/Lee157/finance-skills |
| 分支 | **main** |
| 仓内路径 | `skills/<技能英文id>/`（如 `labor-invoice-check`） |

开发侧：改完 push `main` → GitHub + Gitee。  
**push 了不会自动进你电脑**——本机还要 pull 同步或装新 zip。

## 本机技能装在哪

- macOS/Linux：`~/.config/opencode/skills/`
- Windows：`%USERPROFILE%\.config\opencode\skills\`

## 怎么更新（三选一）

1. **已 clone 上述仓库**：`git pull` → 用新版 `skills/*` **白名单覆盖**到本机 opencode skills（**保留各技能里你改过的 config/**）→ 重启 opencode  
2. **只有 zip**：桌面放 `财务技能包_vX.Y.Z.zip`，用手册/仓库 README 里的「更新提示词」让 opencode 覆盖安装 → 重启  
3. **让 AI 更新**：把本文件路径告诉它，或说「按财务技能包来源与更新说明，从 Gitee/GitHub finance-skills 的 main 更新」
