# 部门费用归集分摊

总账会计每月「按部门归集损益」自动化。**技能包、同事自己拖文件跑，数据只在本机。**

**产出**：`部门科目余额表` + `利润表` + `运行报告.txt`  
**验收**：各部门加总与主体合计核对 ≈ 0（容差见 config）；禁止改数凑平。

## 你（agent）该怎么干

**用户是财务同事。** 她给一个装好当月材料的文件夹，你认文件 → 跑脚本 → 交表。别让她改文件名、看列号。

### 0. 红线
1. **金额只由** `scripts/allocate.py` **计算**，不许心算凑数。  
2. **禁止把金额明细、员工姓名打到对话里**；只说是否成功、缺什么、核对是否有差额科目编码。  
3. **核对不平禁止改数凑平**；报告列编码，停下来问她。  
4. 真实数据不上外网、不进 git。

### 1. 确认材料文件夹
建议子夹（名称近似即可，脚本按内容认）：

| 夹 | 内容 |
|----|------|
| 01_主体余额表 | 甲骨易/文化/上海 等 U8「发生额及余额表」 |
| 02_外地代理报表 | 山东/湖南等代理或出纳表（可选） |
| 03_收入底稿 | 开票+微信支付宝+无票外汇（有申请人、金额列） |
| 04_人员归属表 | 姓名、组织架构1、组织架构2 |
| 05_用友按人明细 | 辅助核算/按人费用（姓名、科目、金额） |
| 06_工资社保 | 若单独导出可放这（同源用友，归集用） |

可先盘点：
```bash
python3 "<本skill目录>/scripts/inspect_inputs.py" --input-dir <材料文件夹绝对路径>
```

### 2. 跑归集
```bash
python3 "<本skill目录>/scripts/allocate.py" \
  --input-dir <材料文件夹绝对路径> \
  --out <同文件夹或桌面>/部门科目余额表_YYYYMM.xlsx
```
- 规则：`config/业务规则.md`  
- **540101、540103** → 整挂「项目总监及助理」  
- **其余 5401\***、**5501/5502/5503** → 按人挂部门  
- **5402/5504** → 财务中心  
- **5101** → 收入三源按营销人→部门  
- 缺关键输入时如实写在运行报告，能算的仍产出。

### 3. 收尾（人话）
- *「部门表已出，在 \<路径\>；运行报告在同目录。请您打开看核对列是否≈0。」*  
- 有未匹配人员/待拆分科目：只报**个数+科目编码**，不报金额人名。  
- 装依赖：`pip install openpyxl xlrd pandas`（国内镜像）。

### 4. 口头改规则 → 只改 config
- 整挂/映射/别名 → `config/` 下 md/json，改完重跑。

## 验收
`python3 tests/test_robustness.py` 应全绿（合成数据）。
