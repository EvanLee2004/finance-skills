# 依赖与安装清单（活表）

> 这是 env-doctor 的本体：**每个技能需要什么、缺了怎么装（国内镜像优先，国外备选）**。
> 维护：李明昊 · 初版 2026-06-24（依据对 8 个技能脚本的真实 `import` 扫描）。**加了新技能，把它的新依赖补到下面对应分组即可。**

镜像优先级（铁律）：**国内优先 → 国内备选 → 国外兜底**。
- pip 国内：清华 `https://pypi.tuna.tsinghua.edu.cn/simple`；备选阿里 `https://mirrors.aliyun.com/pypi/simple`、中科大 `https://pypi.mirrors.ustc.edu.cn/simple`。
- pip 国外兜底：默认源（不加 `-i`）。

## 一、Python 库（pip 装）

### 1. 核心库（脚本真正 import / 调用，全 8 技能基础——优先全装这批）

**一把全装（国内镜像，推荐先跑这条）：**
```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pandas openpyxl pypdf pdfplumber pdf2image python-docx python-pptx markitdown lxml defusedxml Pillow
```
国外兜底：去掉 `-i ...` 部分即可。

| import 名 | pip 包名 | 哪些技能要它 | 备注 |
|-----------|----------|--------------|------|
| `pandas` | **pandas** | receivables-merge / split-by-sales / labor-invoice-check / xlsx | 读写表、算账龄 |
| `openpyxl` | **openpyxl** | 三个财务技能 / xlsx | Excel 读写、样式、下拉框 |
| `pypdf` | **pypdf** | pdf | 合并/拆分/读页 |
| `pdfplumber` | **pdfplumber** | pdf | 抽文字/表格 |
| `pdf2image` | **pdf2image** | pdf（convert_pdf_to_images.py） | ⚠ 依赖系统级 **poppler**（见下） |
| `docx` | **python-docx** | docx | ⚠ 不是 `pip install docx`！ |
| `pptx` | **python-pptx** | pptx | ⚠ 不是 `pip install pptx`！ |
| `markitdown` | **markitdown** | pptx（`python -m markitdown` 读正文）、docx 读取 | 命令行调用 |
| `lxml` | **lxml** | docx / pptx / xlsx（office 解包校验） | XML 处理 |
| `defusedxml` | **defusedxml** | docx / pptx / xlsx（office） | 安全 XML 解析 |
| `PIL` | **Pillow** | pdf / pptx（缩略图） | ⚠ import 叫 PIL，装的是 Pillow |

### 2. PDF 高级库（仅做 OCR / 高级转图 / 用代码造 PDF 时才需要，按需装）

```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pytesseract reportlab pypdfium2
```

| pip 包名 | 用途 | 备注 |
|----------|------|------|
| **pytesseract** | 扫描件 OCR | ⚠ 还需系统级 **tesseract**（见下） |
| **reportlab** | 用代码从零生成 PDF | pdf reference 里的进阶用法 |
| **pypdfium2** | 另一种 PDF 渲染引擎 | 进阶/备选 |

> ⚠ **不要 pip 装的名字**：报错里出现 `validators`、`helpers`、`office` 这些，是技能**自带的本地模块**（`xlsx/scripts/office/validators/` 等），不是 PyPI 包——`pip install` 它们会装错东西。这类报错通常是"没在技能目录里跑/路径不对"，不是缺依赖。

## 二、系统级工具（不是 pip，按系统装）

| 工具 | 哪些技能/功能要 | 缺了会怎样 | Windows 装法（国内优先） | Mac 装法 |
|------|------------------|------------|--------------------------|----------|
| **LibreOffice**（命令 `soffice`） | xlsx 重算公式(recalc)、pptx 转缩略图(thumbnail)、docx 接受修订(accept_changes) | 这几个功能用不了；**openpyxl/python-docx/pypdf 的基本读写不受影响** | `winget install TheDocumentFoundation.LibreOffice`；或国内镜像下载：清华 `https://mirrors.tuna.tsinghua.edu.cn/libreoffice/`、中科大 `https://mirrors.ustc.edu.cn/tdf/libreoffice/` | `brew install --cask libreoffice` |
| **poppler**（命令 `pdftoppm`） | pdf 的 `pdf2image` 把 PDF 转图片 | 报 `Unable to get page count` / poppler 相关错 | scoop `scoop install poppler`；或下 poppler-windows release 加到 PATH | `brew install poppler` |
| **tesseract**（命令 `tesseract`） | pdf 扫描件 OCR（pytesseract） | OCR 报 `tesseract is not installed` | scoop `scoop install tesseract`；或 UB-Mannheim 安装包（含中文需选 `chi_sim` 语言包） | `brew install tesseract tesseract-lang` |
| **Node.js**（命令 `node`/`npm`） | docx/pptx **从零新建**文档（docx-js / pptxgenjs）；只是读/改现有文件不需要 | 从零造 Word/PPT 的进阶路径用不了 | 国内镜像：`npm config set registry https://registry.npmmirror.com`（淘宝镜像）后再装包；Node 本体 winget `winget install OpenJS.NodeJS` 或 nodejs.org | `brew install node` |

## 三、Python 版本

- **建议 Python ≥ 3.10**。低于 3.10（如 3.9）时，office 的可选校验脚本 `*/scripts/office/validate.py` 会因 `match` 语法报 `SyntaxError`。
- **影响范围有限**：这只是"打包前可选校验"那一步；**各技能的核心读写、财务三技能的全部功能在 3.9 也正常**。
- 升级建议：python.org 下载 3.10+（国内可用华为云镜像 `https://mirrors.huaweicloud.com/python/`），Windows 装时勾 **Add to PATH**。

## 四、给 agent 的速查（报错 → 装什么）

| 看到的报错 | 缺什么 | 怎么办 |
|------------|--------|--------|
| `No module named 'pandas'`（或 openpyxl/pypdf/pdfplumber/lxml/defusedxml） | 对应 Python 库 | 跑「一把全装」命令（国内镜像） |
| `No module named 'docx'` | python-docx | `pip install -i <国内镜像> python-docx`（**不是 docx**） |
| `No module named 'pptx'` | python-pptx | `pip install -i <国内镜像> python-pptx` |
| `No module named 'PIL'` | Pillow | `pip install -i <国内镜像> Pillow` |
| `No module named 'pdf2image'` / `Unable to get page count` | pdf2image + poppler | 装 pdf2image（pip）+ poppler（系统） |
| `tesseract is not installed` | tesseract | 装系统级 tesseract |
| `soffice: command not found` / 找不到 soffice | LibreOffice | 装 LibreOffice（见上） |
| `SyntaxError` 指向 `match` | Python 太老 | 升级到 3.10+；或跳过这步可选校验 |
| 缺 `validators`/`helpers`/`office` | （不是缺依赖） | 本地模块，检查是否在技能目录里、用绝对路径调脚本 |
